Beskrivning: Fin

Syfte/Tankegång

Mitt syfte med bloggen är att ge folk möjligheten att lägga upp blogginlägg när de skulle vilja. Just nu så har jag några exempel på bloggar som länkar till andra sidor men tanken var att bloggen skrivs på min hemsida och sedan publiceras i en slags feed för läsare att kolla igenom.

När jag skapade hemsidan så började jag först med att tänka igenom hur designen på hemsidan skulle vara och hur den skulle hänga ihop mellan alla olika sidor. Jag började sedan på att göra headern då den kommer befinna sig på varje sida. Jag använde mig av position: fixed för att få headern fast på toppen samt space-between för att få ett avstånd mellan nav-links och Blogg.

CSS Grid

Grid används i CSS som en layoutmodell. Det grid gör är att den skapar en tvådimensionell rutnät som kan används för att style äns hemsida. Grid har egenskaper såsom grid-template-columns och grid-template-rows för att bestämma strukturen på rutnätet. Man har därefter möjjligheten att använda sig av grid-gap för att kontrollera avståndet mellan alla gridceller.

CSS Flexbox

Flexbox är också en layoutmodell som används i CSS. Flexbox används för att designa och hantera layouter i en webbplats. Några exempel på Flexbox egenskaper skulle kunna inkludera, align-items, justify-content, flex-direction osv. Dessa egenskaper kan justera olika element som befinner sig i en flex-container.
